FastAI.ImagePreprocessing

ImagePreprocessing(means, stds[; augmentations, C = RGB{N0f8}, T = Float32])

Converts an image to a color C, then to a 3D-array of type T and finally normalizes the values using means and stds.

If no means or stds are given, uses ImageNet statistics.
