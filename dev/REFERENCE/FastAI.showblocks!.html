FastAI.showblocks!

showblocks([backend], block, obss)
showblocks!(handle, backend, block, obss)

Show a vector of observations obss of the same block type.

Examples

data, blocks = loaddataset("imagenette2-160")
samples = [getobs(data, i) for i in range(1:4)]
showblocks(blocks, samples)

Extending

This is used for showing batches of observations, unlike the Tuple variant of showblock! which assumes an observation consists of multiple blocks.

Usually, a FastAI.ShowBackend will show an observation in one row with showblock! and showblocks! will show multiple rows.
