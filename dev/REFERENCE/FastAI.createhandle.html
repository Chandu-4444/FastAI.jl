FastAI.createhandle

createhandle(backend::ShowBackend)

Creates a context to which blocks of data can be shown using the mutating functions showblock! and showblocks!. It is called internally when using showblock or showblocks.

handle = createhandle(backend)
showblock!(handle, backend, block, obs)

# Above is equivalent to
showblock(backend, block, obs)
