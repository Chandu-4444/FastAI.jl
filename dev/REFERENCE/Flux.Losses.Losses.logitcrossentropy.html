Flux.Losses.logitcrossentropy

logitcrossentropy(ŷ, y; dims=1, ϵ=eps(ŷ), agg=mean)

Return the crossentropy computed after a Flux.logsoftmax operation; calculated as

agg(.-sum(y .* logsoftmax(ŷ; dims=dims); dims=dims))

logitcrossentropy(ŷ, y) is mathematically equivalent to Flux.Losses.crossentropy(softmax(ŷ), y) but it is more numerically stable.

See also: Flux.Losses.crossentropy, Flux.Losses.binarycrossentropy, Flux.Losses.logitbinarycrossentropy
