FastAI.ShowBackend

abstract type ShowBackend

Abstract type for backends that allow showing blocks of data in an interpretable way.

Extending

For a ShowBackend Backend, you should implement the following methods:

  • FastAI.createhandle(::Backend) creates a context that blocks of data can be shown to

  • FastAI.showblock!(handle, ::Backend, block::B, obs) shows a block of type B. This needs to be implemented for every block type you want to be able to show

  • FastAI.showblocks!(handle, ::Backend, blocks, obss) shows a collection of blocks
