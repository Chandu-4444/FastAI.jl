FluxTraining.Phases.ValidationPhase

ValidationPhase()

A regular validation phase. It iterates over batches in learner.data.validation and performs a forward pass.

Throws the following events: FluxTraining.Events.EpochBegin, FluxTraining.Events.StepBegin, FluxTraining.Events.LossBegin, FluxTraining.Events.StepEnd, FluxTraining.Events.EpochEnd.

Throws the following events in this order:

  • FluxTraining.Events.EpochBegin when an epoch starts,

  • FluxTraining.Events.StepBegin when a step starts,

  • FluxTraining.Events.LossBegin after the forward pass but before loss calculation,

  • FluxTraining.Events.StepEnd when a step ends; and

  • FluxTraining.Events.EpochEnd when an epoch ends

It writes the following step state to learner.state, grouped by the event from which on it is available.

  • StepBegin:

    • xs and ys: encoded input and target (batch)

  • LossBegin:

    • yÌ‚s: model output

  • StepEnd:

    • loss: loss
